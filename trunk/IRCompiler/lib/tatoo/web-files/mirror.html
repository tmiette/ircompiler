How-to make a Tatoo mirror

To create a copy of Tatoo web site, simple follow these three steps:

  • download project archive tatoo.zip
  • extract it in a temporary directory root
  • run ant web in directory root/web; the script will ask you where to install the site. This location can be written in file root/web/local.properties (see sample file included for format, use / and not \ for path separation).
